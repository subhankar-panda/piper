[ piper ]
Command line outputs to the web, one pipe away!

Piper is a command line tool to easily and seamlessly share command line outputs to the web, by using a unix pipe to the piper command.

You can get piper by running the following commands in a unix shell : 

 
$ git clone https://github.com/subhankar-panda/piper.git piper
$ cd piper
$ go install -v

I'm working on getting it up on npm/brew <\shrug>

Once you have piper installed, you can get to using it like this : 


$ python frustating_err_code.py > err_python
$ cat err_python | piper

Piper will generate a ~sick~ URL for you and host your file on the web for you to share your findings/errs, straight from your shell!
